# Usability Test (Peer Session)

## Tasks
1) Find the Library using search.
2) Get directions from Main Gate to CS Building.
3) Change destination to Cafeteria and re-run directions.

## Metrics
- Time to complete each task
- Number of mis-clicks / wrong selections
- Self-reported confidence after task completion

## Findings (example)
- Users prefer autocomplete suggestions.
- Clear feedback improves confidence (status text area).
- Visible focus ring helps keyboard-only users.
