# University Library Management System (Database)

Relational database design for a university library (books, members, borrowing records, and fines). Includes normalization notes and SQL scripts for SQL Server and MySQL.

## Context
- Academic year: 2nd year
- Course: Database Management
- Developed during: January 2023

## Included
- ERD (Mermaid) in docs/erd.md
- SQL Server scripts in sqlserver/
- MySQL scripts in mysql/

## Entities
- Books
- Members
- Loans (borrow/return tracking)
- Fines (auto-created for overdue returns)
